# 📐 Calculador de Geometría - Triángulos Isósceles

Este proyecto es una aplicación de consola desarrollada en **Java** diseñada para la gestión y análisis de figuras geométricas. El programa permite trabajar con múltiples instancias de objetos, realizar cálculos matemáticos complejos y procesar colecciones de datos de forma eficiente.

### 🖥️ Funcionalidades principales:
* **Modelado POO:** Implementación de clases con encapsulamiento (uso de atributos privados y métodos de acceso).
* **Cálculos Geométricos:** Algoritmos para obtener área y perímetro utilizando la librería `Math` (Teorema de Pitágoras aplicado).
* **Lógica de Negocio:** Implementación de un algoritmo de búsqueda para determinar la figura con mayor superficie dentro de un array de objetos.
* **Interfaz Dinámica:** Gestión de entrada de datos interactiva mediante la clase `Scanner` y salida formateada para mayor claridad visual.

### 🛠️ Tecnologías utilizadas:
* **Lenguaje:** Java 
* **Paradigma:** Programación Orientada a Objetos (POO)
* **Entorno:** Compatible con cualquier IDE (IntelliJ, Eclipse, VS Code)
